
# Orientador de Disciplinas - BICT para Eng. Comp

Este projeto consiste em uma aplicação em Java com interface gráfica desenvolvida utilizando o pacote java.swing, cujo objetivo é auxiliar na organização e orientação das disciplinas do curso de Engenharia da Computação.

O sistema permite realizar o cadastro, alteração e exclusão de disciplinas, aplicando conceitos de Programação Orientada a Objetos, como herança, polimorfismo e uso de collections, além de armazenamento permanente em arquivo.

## Funcionalidades

- Cadastro de disciplinas obrigatórias e optativas

- Alteração de disciplinas já cadastradas

- Exclusão de disciplinas

- Listagem das disciplinas cadastradas

- Exibição da carga horária calculada a partir dos créditos

- Controle do limite máximo de 27 créditos por período

- Validação para impedir códigos de disciplinas repetidos

- Validação para exigir que o código da disciplina tenha exatamente 8 caracteres

## Conceitos Utilizados

- Programação Orientada a Objetos

- Herança

- Polimorfismo

- Collections (ArrayList)

- Interface gráfica com Swing

- Manipulação de arquivos para persistência de dados
